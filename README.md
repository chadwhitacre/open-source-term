Scripts used to produce the magazine and Usenet charts in "[OSI Really Did Initiate Open Source](https://openpath.quest/2025/osi-really-did-initiate-open-source/)," shared under the [CRAPL](https://github.com/chadwhitacre/open-source-term/blob/main/CRAPL-LICENSE.txt).

Sketch of my method:

1. Download datasets.
2. Wrestle into SQLite.
3. Run full-text search queries suggested by ChatGPT.
4. Dump to CSV.
5. Load into [Google Sheets](https://docs.google.com/spreadsheets/d/1DumpVEKxUU9wm6LBoVFIRG63atdw_EubKCNwHq3mvP4/edit?usp=sharing), generate chart, export as PNG, add OSI logo in Pixelmator.
